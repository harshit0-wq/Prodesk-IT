# Prodesk IT — Digital Marketing Landing Page (Sprint 01)

A responsive and modern landing page built for the Digital Marketing division of Prodesk IT. This project was developed as part of Sprint 01 to demonstrate frontend development fundamentals, responsive design practices, and user-focused interface design.

## Live Deployment

**Live Demo:** [https://prodesk-landing-page-theta.vercel.app/]

---

## Features

### Core MVP

* Responsive navigation bar with desktop and mobile support
* Hero section with headline, supporting text, and call-to-action buttons
* Services section featuring:

  * Web Development
  * SEO Optimization
  * Digital Marketing
* Footer with copyright information and social media links

### UI/UX Enhancements

* Dark/Light theme toggle using JavaScript and localStorage
* Hover animations for buttons and service cards
* Sticky navigation for improved user experience
* Smooth transitions and interactive visual feedback

### Additional Improvements

* Tailwind CSS utility-first styling
* Glassmorphism-inspired navigation design
* Mobile-first responsive approach
* Accessibility-conscious color and typography choices
* Inline SVG logo for optimized loading performance

---

## Technical Highlights

### Responsive Design

The interface is designed to work across desktop, tablet, and mobile devices using modern CSS and Tailwind CSS utilities.

### Theme Persistence

The selected theme is stored in localStorage, ensuring user preferences persist across sessions.

### Performance Optimization

The logo is implemented as an inline SVG, reducing external asset requests and helping improve page load performance.

### Accessibility

Care was taken to maintain readable typography, sufficient spacing, and accessible color contrast throughout the interface.

## Testing

To evaluate performance and accessibility using Google Lighthouse:

1. Open https://prodesk-landing-page-theta.vercel.app/.
2. Press `F12` to open DevTools.
3. Navigate to the **Lighthouse** tab.
4. Select:

   * Device: Desktop
   * Mode: Navigation
5. Run the audit and review the generated report.
