# AI Prompt Engineering & Development Log

This document records the major prompts and development decisions used during the implementation of the Prodesk IT Digital Marketing Landing Page for Sprint 01.

## Prompt 01: Initial Project Setup

**User Prompt:**

> "Welcome to your first active Sprint at Prodesk IT... provide me with everything, provide me with full code of all phases in one heading with different phases and tasks in respective phases."

**Purpose:**

* Understand project requirements
* Create the initial landing page structure
* Implement the required Phase 1 features
* Set up the project using Tailwind CSS
* Organize all deliverables within a single HTML file

---

## Prompt 02: Theme Customization

**User Prompt:**

> "theme colour yellow and white"

**Purpose:**

* Apply the requested brand color scheme
* Update the design to use yellow and white as primary colors
* Improve visual consistency across all sections
* Ensure text remains readable with appropriate color contrast

---

## Prompt 03: External Link Integration

**User Prompt:**

> "https://www.linkedin.com/company/prodesk-it add this LinkedIn URL and logo ... https://prodesk.in/ add this link to logo"

**Purpose:**

* Connect navigation elements to official company resources
* Link the website logo to the Prodesk IT homepage
* Add the official LinkedIn page to social media links
* Replace placeholder links with actual destinations

---

## Prompt 04: Inline Logo Implementation

**User Prompt:**

> "isnt there any way i can save the image in one code file index.html so it will be you know one file easy peasy ... also image directly in code"

**Purpose:**

* Remove dependency on external image files
* Convert the logo into inline SVG markup
* Keep the project self-contained within a single HTML file
* Simplify deployment and file management

**Outcome:**

* Eliminated the need for a separate logo image file
* Reduced external asset dependencies
* Improved project portability and ease of deployment

---

## Summary

The final implementation evolved through multiple iterations focused on layout creation, theme customization, link integration, and asset optimization. The resulting project is a responsive single-file landing page built using HTML, Tailwind CSS, JavaScript, and inline SVG assets.
